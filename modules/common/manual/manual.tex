\paragraph{}
\en{
During setup, BUMM can be configured with different countdown settings and a manipulation detector (MD).
The MD submodule can activate the connected BUMM effector if an unauthorized manipulation of the modules is detected.
It can be configured with different sensitivities:
Low sensitivities will only trigger the manipulation alert if multiple mistakes are made during defusal.
As long as at least one of the MD health indicators (above the countdown timer) is lit, the MD is in safe mode.
If all indicator LEDs are off, any mishandling of the modules will immediately activate the MD alert and the connected effector.
}
\de{
Während der Konfiguration können verschiedene Countdown-Zeiten sowie ein Manipulationsdetektor (MD) eingestellt werden.
Der MD kann den angeschlossenen Effekt aktivieren, wenn eine unauthorisierte Manipulation der Module erkannt wird.
Der MD kann verschieden empfindlich eingestellt werden:
Mit niedriger Empfindlichkeit wird der MD erst aktiviert, wenn mehrere Fehler während der Entschärfung passieren.
Solange mindestens eine der MD-Sicherheitsleuchten (oberhalb des Countdowns) leuchtet, ist der MD sicher.
Erst wenn alle Leuchten aus sind, führt die nächste Fehlbedienung eines Moduls zu einer Aktivierung des MD und des Effektors.
}
\chde{
Obwohl die Konfiguration verschiedener Countdown-Zeit und Manipulationserkennung (MD) eingestellt werden.
Wenn das Detektionsmodul MD unbefugte Manipulation der Effekt aktivieren kann angeschlossen ist.
MD können unterschiedliche Empfindlichkeit eingestellt:
Mit geringer Empfindlichkeit MD nur, wenn treten mehrere Fehler gleichzeitig aktiviert entschärfen.
Solange die MD- Sicherheitslampe (Countdown oder mehr) mindestens eine leuchtet, ist MD sicher.
Erst wenn alle Lichter aus sind, ein Modul in einer falschen Bedienung Aktivierung führt MD und Effekte.
}
